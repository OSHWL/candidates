OSHWL candidates

documents not yet "selected"

please try to have the documents meet the following criteria:

  1. possible formats:
    • if not exceeding 50 lines: Markdown
    • LaTeX (unique class to be determined) in order to generate PDF and/or EPUB files (in SELECTION, each document's PDF and EPUB files will be made available side by side, ready to be read)
  2. english language
  3. subjective qualities such as completeness, readability and uniformity among others.

please upload all the documents's source files to the same folder as the document itself, to allow collaboration.

big categories of OSHW work in which documents could be found:

  • applications in which the hardware in question tends to be used:

    • energy
    • transport
    • agriculture
    • water
    • household
    • construction

  • know-how/knowledge that should be useful in any OSHW work:

    • manufacturing
    • systems
    • OSHW management
    • electro
    • free/libre/open-source software for OSHW
    • materials
    • acoustics
    • control
